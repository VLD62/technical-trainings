# Lab 04 - Procedures, Functions, and Packages

## Goal

Use the stored APIs installed by the lab and extend them safely.

## Tasks

### 1. Annual Salary

Call the annual-salary function for employee 1 on both platforms.

### 2. Raise Procedure

Start an explicit transaction, call the raise procedure with 5 percent, inspect the new salary, and roll back.

### 3. Input Validation

Call the procedure with:

- A negative percentage
- A percentage above the allowed maximum
- A missing employee ID

Record the error contract on each platform.

### 4. Department Payroll

Create a function or procedure that returns total payroll for a department.

Requirements:

- Validate that the department exists.
- Define the result for a department with zero employees.
- Avoid a cursor.

### 5. Oracle Package Review

Identify which objects are public in the package specification and which implementation details remain in the body.

### 6. SQL Server Organization Review

Explain how schemas, permissions, and naming conventions can group related SQL Server APIs without providing a direct package equivalent.
