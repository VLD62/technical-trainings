# Quiz 03 — Provider Comparison and Operations

## Questions

1. What is the AWS virtual networking service?
2. What is the Azure workload billing and quota boundary commonly called?
3. What is the central Google Cloud workload and API boundary?
4. What OCI construct organizes resources and permissions inside a tenancy?
5. Name the managed Kubernetes service for each of the four providers.
6. Why is a provider service map only a starting point?
7. What is Infrastructure as Code?
8. What is policy as code?
9. What is a landing zone?
10. What is the purpose of mandatory ownership metadata?
11. Name three common cloud cost drivers that teams underestimate.
12. What is a unit-economics metric?

## Answers

1. Amazon VPC.
2. Subscription.
3. Project.
4. Compartment.
5. Amazon EKS, Azure Kubernetes Service, Google Kubernetes Engine, and Oracle Kubernetes Engine.
6. Similar categories can have different architecture, limits, regions, pricing, security, and operations.
7. Version-controlled definitions used to provision and configure infrastructure.
8. Automated evaluation or enforcement of configuration rules.
9. A reusable foundation for identity, networking, governance, logging, security, billing, and workload deployment.
10. It enables accountability, lifecycle management, governance, and cost allocation.
11. Network egress, logs and retention, backups, idle non-production resources, public IPs, and premium support are valid examples.
12. A cost measurement linked to business output, such as cost per order or cost per API request.
