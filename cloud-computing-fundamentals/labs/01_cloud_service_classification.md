# Lab 01 — Cloud Service Classification

## Objective

Classify common technology offerings by deployment model and service model.

## Tasks

For each scenario, identify the most likely model: on-premises, IaaS, PaaS, SaaS, CaaS, DBaaS, or FaaS. Then state the main customer responsibilities.

1. A company rents a virtual machine and installs its own database.
2. A team deploys source code to a managed web application runtime.
3. Employees use a hosted collaboration suite in a browser.
4. A function processes each object uploaded to a bucket.
5. A managed PostgreSQL service performs platform patching and automated backups.
6. A managed Kubernetes cluster provides the control plane.
7. An internal OpenStack environment offers self-service VMs through APIs.

## Deliverable

Create a table with:

- Scenario
- Deployment model
- Service model
- Provider responsibilities
- Customer responsibilities
- Main benefit
- Main risk

## Reflection

- Which scenarios are commonly misclassified?
- Which managed services still require substantial customer operations?
- Where does the shared-responsibility boundary remain ambiguous?
