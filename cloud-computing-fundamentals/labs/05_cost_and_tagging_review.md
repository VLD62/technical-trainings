# Lab 05 — Cost and Tagging Review

## Objective

Create a cost ownership and optimization plan for ShopNow.

## Inputs

- `examples/tagging-policy.yaml`
- ShopNow workload requirements
- Provider-neutral architecture from Lab 02

## Tasks

1. List all likely cost categories.
2. Identify fixed, usage-based, and peak-driven costs.
3. Define mandatory tags or labels.
4. Define a monthly budget and alert thresholds.
5. Define an anomaly-detection response.
6. Identify five waste scenarios.
7. Define a non-production shutdown or expiry policy.
8. Select two unit-economics metrics.

## Required Cost Categories

- Compute
- Database
- Object and block storage
- Network egress
- Load balancing and public endpoints
- Logs, metrics, and traces
- Backups and replicas
- Support or licenses

## Deliverable

Create:

- Tagging standard
- Budget alert matrix
- Monthly cost review checklist
- Optimization backlog ordered by expected value and risk

## Reflection

Explain why the lowest infrastructure price might not produce the lowest total cost of ownership.
