# Bash Fundamentals Labs

Complete the labs in order. Use a disposable working directory and never point destructive commands at production data.

## Workflow

For every lab:

1. Read the task and acceptance criteria.
2. Create the requested script in your own working directory.
3. Test with normal input, empty input, invalid input, and paths containing spaces.
4. Run `bash -n your_script.sh`.
5. Run ShellCheck when available.
6. Compare the result with the related example only after attempting the task.
