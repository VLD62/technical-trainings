# Exam 102 Review and Final Practice

### Consolidation for topics 105–110

**Presenter:** Vladislav Iliev

**Audience:** Linux learners, system administrators, DevOps engineers, and certification candidates

---

## High-Weight Areas

- Users and groups, job scheduling, networking, scripting, logging, and encryption deserve repeated practice.
- Know both modern tools and legacy terms still present in the objectives.
- Understand configuration files as well as commands.

---

## Mixed Scenario Practice

- A scheduled report did not run.
- A user cannot log in after an account change.
- A service is listening unexpectedly.
- The clock is unsynchronized.
- DNS resolution fails while direct IP connectivity works.

---

## Security Scenarios

- Find SUID and SGID files.
- Review active sessions and listeners.
- Create and use an SSH key pair.
- Encrypt, decrypt, sign, and verify a file with GPG.
- Apply a narrow sudo rule safely.

---

## Scripting Scenarios

- Validate arguments.
- Read a file line by line.
- Check a command exit status.
- Use a loop and conditional.
- Produce deterministic output with locale control.

---

## Exam Technique

- Use the exact objective wording as a study guide.
- Practice fill-in-the-blank command names and paths.
- Watch for commands that differ by distribution.
- Time-box uncertain questions and return later.

---

## Final Readiness

- Complete both checklists.
- Score consistently on mixed quizzes.
- Complete the practical final exam without destructive shortcuts.
- Review the official objectives immediately before scheduling the exam.

---

## Next Steps

- Revisit weak objectives by weight.
- Build a fresh VM from installation to secured networked host.
- Maintain a concise personal command sheet.
- Book each exam only when practical recall is reliable.

---

## Practice

- Run the commands in a disposable lab.
- Capture expected and unexpected output.
- Explain what changed and how to reverse it.
- Complete the matching lab and checklist items.

---

## Key Takeaways

- Understand the concept before memorizing options.
- Know the important commands, files, and failure modes.
- Prefer safe, observable, reversible practice.
- Review the official LPIC-1 objectives as the source of truth.
