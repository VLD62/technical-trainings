# Linux Administration: Basics

### Course introduction and LPIC-1 roadmap

**Presenter:** Vladislav Iliev

**Audience:** Linux learners, system administrators, DevOps engineers, and certification candidates

---

## Why Learn Linux Administration?

- Linux is a foundation for cloud, DevOps, platform engineering, cybersecurity, embedded systems, and enterprise infrastructure.
- Administration skills are built through repeated command-line practice, not memorization alone.
- LPIC-1 is vendor-neutral and requires knowledge across multiple Linux families.

---

## LPIC-1 Certification Path

- LPIC-1 version 5.0 consists of exams 101-500 and 102-500.
- Both exams must be passed to earn the certification.
- Each objective has a weight; higher-weight objectives deserve more practice.

---

## Course Philosophy

- Understand the concept.
- Recognize the relevant files and commands.
- Practice in a disposable environment.
- Troubleshoot deliberately.
- Explain the result in your own words.

---

## Recommended Lab

- Ubuntu Server LTS for Debian-family tasks.
- Rocky Linux or AlmaLinux for RPM/DNF tasks.
- Optional openSUSE for Zypper awareness.
- Snapshots before risky changes.

---

## Learning Workflow

- Slides → command practice → lab → quiz → checklist.
- Keep a command journal with examples and common errors.
- Repeat high-weight objectives until the commands are automatic.

---

## Course Boundaries

- The course aligns with published objectives but does not reproduce official exam questions.
- Use the official objectives as the source of truth.
- Review objective updates before booking an exam.

---

## Practice

- Run the commands in a disposable lab.
- Capture expected and unexpected output.
- Explain what changed and how to reverse it.
- Complete the matching lab and checklist items.

---

## Key Takeaways

- Understand the concept before memorizing options.
- Know the important commands, files, and failure modes.
- Prefer safe, observable, reversible practice.
- Review the official LPIC-1 objectives as the source of truth.
