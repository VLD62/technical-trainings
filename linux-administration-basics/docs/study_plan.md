# LPIC-1 Study Plan

This plan assumes approximately 8–10 hours per week. Adjust it to your experience and available time.

## Eight-Week Plan

| Week | Focus | Deliverable |
|---|---|---|
| 1 | Course setup, hardware, boot, targets | Complete Lab 1 and explain the boot sequence |
| 2 | Disks, filesystems, GRUB, package management | Complete Labs 2–3 on Debian and RPM systems |
| 3 | Command line, files, archives, text filters | Complete Labs 4–5 without notes |
| 4 | Processes, permissions, links, FHS | Complete Labs 6–7 and Exam 101 quiz |
| 5 | Shell environment and scripting | Complete Lab 8 and write two original scripts |
| 6 | Users, groups, scheduling, locale, services | Complete Labs 9–10 |
| 7 | Networking, DNS, security, SSH, GPG | Complete Labs 11–12 |
| 8 | Mixed review, checklists, practical exam | Complete both quizzes and final practical |

## One-Week Intensive Review

Use this only when you already have Linux administration experience.

| Day | Focus |
|---|---|
| 1 | Objectives 101 and 102: architecture, boot, storage, packages |
| 2 | Objective 103: commands, text, files, processes, regex, vi |
| 3 | Objective 104 and full Exam 101 practice |
| 4 | Objectives 105–107: shell, scripting, users, jobs, locale |
| 5 | Objectives 108–109: services, logs, time, networking, DNS |
| 6 | Objective 110 and full Exam 102 practice |
| 7 | Weak areas, timed mixed tests, official objective review |

## Daily Routine

1. Review one objective and its weight.
2. Write commands from memory.
3. Run them in the lab.
4. Break one configuration deliberately and recover it.
5. Record mistakes and the reason for each mistake.
6. Answer 10–15 mixed questions.

## Evidence of Readiness

You are approaching readiness when you can:

- Explain why a command is correct and why alternatives are wrong
- Recognize common output and configuration files
- Switch between Debian and RPM-family package workflows
- Complete storage and networking tasks safely
- Write a small Bash script without searching for basic syntax
- Troubleshoot using evidence instead of random command changes
- Score consistently on mixed practice sets under time pressure
