## Study Prioritization

Weights indicate relative emphasis, not a guarantee of exact question distribution. Prioritize:

1. Objectives with weights 4–5
2. Commands and files that appear across multiple objectives
3. Topics that require distribution-specific comparison
4. Weak areas revealed by labs and quizzes

## Maintenance Note

Certification objectives can change. Before using this material for an exam attempt, compare this file with the official objective page and update commands, files, or terminology where required.
