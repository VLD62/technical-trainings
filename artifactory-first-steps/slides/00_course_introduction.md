# JFrog Artifactory: First Steps

## Course Introduction

---

## What this training covers

* What JFrog Artifactory is
* Why artifact repositories are used
* Repository types and artifact lifecycle
* Basic upload, download, and publish workflows
* Access control and good practices
* How Artifactory fits into CI/CD pipelines

---

## Target audience

* Developers
* DevOps engineers
* Build and release engineers
* QA engineers
* Platform and infrastructure engineers
* Technical leads who work with build artifacts

---

## Prerequisites

Participants should know the basics of:

* Git
* Build outputs and package files
* CI/CD concepts
* Command-line usage

---

## Training format

* Short theory sections
* Practical workflow examples
* Generic commands using placeholders
* Discussion points for team-specific usage

---

## Repository layout

```text
artifactory-first-steps/
```

---

## Learning goal

After this training, participants should be able to explain where artifacts belong, how they move through the delivery lifecycle, and how to safely upload and retrieve them from Artifactory.
