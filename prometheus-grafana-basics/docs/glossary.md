# Glossary

## Alert

A condition evaluated from metrics that may require attention.

## Annotation

Human-readable context attached to an alert or dashboard.

## Cardinality

The number of unique time series created by combinations of metric names and label values.

## Counter

A cumulative metric that normally increases.

## Data Source

A backend queried by Grafana, such as Prometheus.

## Exporter

A component that exposes metrics for a system in Prometheus format.

## Gauge

A metric that can increase or decrease.

## Histogram

A metric type that groups observations into buckets and records count and sum.

## Instance

A single Prometheus scrape target, normally represented by the `instance` label.

## Job

A logical group of Prometheus targets.

## Label

A key-value attribute used to identify and filter time series.

## Metric

A numeric measurement recorded over time.

## Node Exporter

The Prometheus exporter for Linux hardware and operating-system metrics.

## Prometheus

A monitoring system and time-series database that collects and queries metrics.

## PromQL

The Prometheus query language.

## Recording Rule

A rule that precomputes and stores the result of a PromQL expression.

## Scrape

A Prometheus HTTP request to collect metrics from a target.

## Target

An endpoint that Prometheus attempts to scrape.

## Time Series

A metric name and unique label set with samples over time.
