# Exercise 3 – Grafana Dashboard

## Goal

Create a reusable dashboard for multiple Linux hosts.

## Tasks

1. Create a new dashboard.
2. Add an `instance` variable.
3. Add a target-status panel.
4. Add CPU and memory time-series panels.
5. Add disk free percentage.
6. Configure correct units.
7. Add useful legends.
8. Export the dashboard JSON.

## Design Review

Check:

- Dashboard title
- Variable behavior
- Panel units
- Threshold meaning
- Missing data behavior
- Time range
- Refresh interval
