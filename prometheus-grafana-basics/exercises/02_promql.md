# Exercise 2 – PromQL

## Goal

Write PromQL queries for infrastructure and application monitoring.

## Tasks

Create queries for:

1. All healthy targets
2. Count of targets by job
3. CPU utilization by instance
4. Memory utilization by instance
5. Disk free percentage by mountpoint
6. Application requests per second
7. Application error percentage
8. 95th percentile request duration

## Challenge

Create a query that returns only Linux hosts with memory utilization above 80%.

## Review

For every query explain:

- Metric type
- Labels preserved
- Aggregation used
- Selected time window
