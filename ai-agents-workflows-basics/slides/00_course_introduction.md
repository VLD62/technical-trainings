# 🧠 AI Agents & Workflows: Basics

### Introduction

**Presenter:** Vladislav Iliev  
**Audience:** Developers, DevOps Engineers, IT Professionals

---

# About This Course

This course provides a practical introduction to the world of AI Agents and Workflows.

The goal is to understand:

- What AI Agents are
- How modern AI systems work
- How agents use tools and memory
- How workflows and automation fit into the picture
- The most common technologies and frameworks in the ecosystem

---

# Why This Course?

Artificial Intelligence is evolving from simple chat interfaces to systems that can:

- Reason
- Use tools
- Access external systems
- Execute workflows
- Collaborate with other agents

Understanding these concepts is becoming increasingly important for modern software professionals.

---

# Learning Objectives

By the end of this course you will be able to:

✅ Understand the difference between an LLM and an AI Agent

✅ Explain the role of tools, memory and prompts

✅ Understand workflow automation concepts

✅ Understand the purpose of frameworks such as LangChain and LangGraph

✅ Build a simple AI Agent

# Prerequisites

Recommended knowledge:

- Basic programming skills
- Basic understanding of APIs
- Basic understanding of software architecture

No previous AI experience is required.

---

# Course Philosophy

This is a practical course.

We will focus on:

- Understanding concepts
- Learning the terminology
- Exploring real examples
- Building simple working solutions

The goal is not to become AI experts, but to understand the foundations.
