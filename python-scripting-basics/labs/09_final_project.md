# Lab 9: Final Project — Log Analyzer

## Tasks

1. Accept a UTF-8 log file path.
2. Parse timestamp, level, component, and message.
3. Count malformed lines.
4. Filter by level and aggregate by level and component.
5. Produce text or JSON output with useful exit codes and tests.

## Stretch Task

Add date filtering, directory processing, or threshold-based exit codes.

## Review

- Which responsibility belongs in a function or class?
- Which errors are expected user errors?
- How will the behavior be tested?
