# Lab 8: Testing and Code Quality

## Tasks

1. Implement a timeout parser accepting 0.1 through 60 seconds.
2. Test integers, decimals, boundaries, invalid values, and empty input.
3. Use Arrange, Act, Assert and `assertRaises`.
4. Run `unittest`, `compileall`, and Ruff.
5. Add type hints and a docstring.

## Stretch Task

Make minimum and maximum configurable.

## Review

- Which responsibility belongs in a function or class?
- Which errors are expected user errors?
- How will the behavior be tested?
