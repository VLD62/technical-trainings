# Lab 1: First Script and Environment

## Tasks

1. Create and activate `.venv`.
2. Run `scripts/validate_environment.py`.
3. Extend the system report with username, current directory, Python executable, and platform.
4. Add a `--compact` flag with `argparse`.
5. Return exit code `0` on success.

## Stretch Task

Write the report to a path supplied through `--output`.

## Review

- Which responsibility belongs in a function or class?
- Which errors are expected user errors?
- How will the behavior be tested?
