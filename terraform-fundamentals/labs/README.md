# Terraform Fundamentals Labs

The labs build on one another but can also be completed independently.

| Lab | Topic | Main example |
|---:|---|---|
| 1 | First configuration and core workflow | `examples/01-local-basics` |
| 2 | Variables, collections, locals, and outputs | `examples/02-variables-and-collections` |
| 3 | Build and use a module | `examples/03-local-module` |
| 4 | State and workspaces | `examples/01-local-basics` |
| 5 | Lifecycle, import concepts, and refactoring | `examples/04-lifecycle-and-conditions`, `examples/05-refactoring` |
| 6 | Tests, security, and CI | `examples/06-tests` |

## Lab Rules

- Read the configuration before running commands
- Predict the plan
- Use only disposable local state
- Do not commit generated state, plans, or credentials
- Clean up after every lab
- Record unexpected behavior and the command output that explains it
